# Project: omat-sivut (Automation Portfolio)

Modern software engineering portfolio focused on Automation, Data, and AI as tools. Built with Next.js 15, Tailwind v4, and Framer Motion.

## Vision
To demonstrate a privacy-first, high-performance professional identity that prioritizes "Automation-first" engineering principles.

## Core Tech Stack
- **Frontend**: Next.js 15 (App Router), React 19
- **Styling**: Tailwind CSS v4 (Zero-runtime)
- **Animations**: Framer Motion
- **Security**: Client-side obfuscation for personal data
- **Deployment**: Vercel

## Status
- **Phase 1-2**: Foundation & Privacy (Completed)
- **Phase 3**: Content & Detail Pages (Completed)
- **Phase 4**: Quality, Testing & Performance (IN PROGRESS)
