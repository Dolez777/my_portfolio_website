# Modern Engineering Portfolio

A high-end, motion-driven personal portfolio website built with **Next.js 15**, **React 19**, and **Tailwind CSS v4**. Designed for software engineers specializing in web apps, data automation, and AI integrations.

## ✨ Key Features

- **Modern Tech Stack**: Built with the latest Next.js 15 (App Router) features for maximum performance.
- **Dynamic Design System**: Custom glassmorphism components, HSL-based dark mode, and geometric gradients.
- **Interactive Project Gallery**: Filterable archive using Framer Motion for smooth layout transitions.
- **Responsive & Accessible**: Fully optimized for mobile, tablet, and desktop with semantic HTML and accessible UI primitives (Radix UI).
- **SEO Ready**: Optimized metadata and title structures for every page.

## 🚀 Getting Started

1. **Clone and Install:**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   npm install
   ```

2. **Configure Environment:**
   Copy `.env.example` to `.env.local` and fill in your personal details:
   ```bash
   cp .env.example .env.local
   ```

3. **Run Development Server:**
   ```bash
   npm run dev
   ```

## 🛠️ Built With

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/) inspired Radix primitives
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Space Grotesk (Headings) & Inter (Body)
